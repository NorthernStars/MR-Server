What is Mixed-Reality Gameserver

The NorthernStars Mixed-Reality Gameserver (MR-Server) is a java-based software to manage different game szenarios for the Mixed-Reality system. The Mixed-Reality is a robot soccer league that plays with a set of tiny robots an a simulated soccer field. The robots are controlled remotely by game server wich interacts with several autonomous agents (clients). This server gives you the aibility to start and manage a soccer game as well as any other scenario you can think off (of course you have to program it yourself).
